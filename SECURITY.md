# Security Policy

## Supported Versions

We currently support the following versions of HiSCaM:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not** report it via public issues. Instead, please follow these steps:

1. Send an email to the maintainer (if provided) or open a private draft security advisory on GitHub.
2. Provide a detailed description of the vulnerability, including:
   - The type of vulnerability.
   - The potential impact.
   - Steps to reproduce (including code/prompts).
   - Any suggested fixes.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.

## Responsible Disclosure
We ask that you follow responsible disclosure guidelines and give us reasonable time to address the issue before making it public.
